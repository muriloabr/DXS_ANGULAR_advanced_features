Desenvolvimento de WebSystems com Angular no front-end

Vamos te entregar a melhor plataforma com total segurança!

PRODUTOS #1

Uso de roteamento, configuração de exibição de valores monetários, consumo de dados por JSon, através de um serviço de end-point.

ACESSE ESTE EXEMPLO

CADASTRO #1

Exemplo de cadastro que abrange um formulário reativo, com validações de campos e três bibliotecas externas: nGBrazil, ng2-validation para validações e validações customizadas, e a js-brasil que auxilia no uso de máscaras de campos brasileiros. A exibição de erros é feita de forma individual por cada erro, criando-se um enorme HTML.

ACESSE ESTE EXEMPLO

CADASTRO #2

Exemplo de cadastro que abrange o CADASTRO #1 e troca toda a exibição de validação para um esquema mais limpo utilizando a assinatura dos eventos blur dos componentes para captar numa lista pre-definida as mensagens de validação que o erro correspondente requer que exiba.
A codificação fica mais limpa.
